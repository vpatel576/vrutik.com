Price Reminder Tool

April 2020-June 2020

This tool was built for IMA Portfolio specifically. The script goes through all the holdings and compare the current price with the target price. If the current price is within 3% if the target price or above the target price, an email is sent out to the IMA members notifiying them of these holdings everyday at 3:30 PM.

Tools used: Python
